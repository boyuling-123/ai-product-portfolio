Business Case

PPC 视频生成工作台

面向商品短视频生产，把商品信息、脚本、分镜、分镜图、视频片段、剪辑合成、保存导出串成一个 Agent 可生成、用户可干预、结果可沉淀的工作台。

AI 视频脚本 / 分镜 / 视频片段可编辑工作台

主链路

商品输入商品图、链接、语言、类型、渠道尺寸
脚本生成视频概览、卖点、分镜描述、VO 旁白
分镜画布分镜参考图、生成模式、结果选择
视频片段画面描述、时长、生成结果、高清处理
剪辑导出合成、字幕、配音、素材库、保存下载

项目描述

PPC 视频生成工作台解决的是“AI 能生成视频，但业务生产过程不可控”的问题。方案把一次性对话生成拆为多个可编辑模块，让用户能在脚本、分镜、视频片段和剪辑环节介入，并让过程数据进入项目和资产体系。

需求与设计

  • 入口与入参：定义商品图片、商品链接、目标语言、视频类型、投放渠道及尺寸等字段，支持进入 Agent 后自动唤起固定任务消息并返回前端组件。
  • 脚本环节：设计视频概览、分镜描述、VO 旁白等模块，并支持编辑、重新生成、双语辅助和适配视频时长翻译。
  • 分镜环节：设计未生成 / 已生成分镜状态，支持分镜参考图、首帧参考、首尾帧参考、多图参考、重新生成、局部修改、替换、复制和删除。
  • 视频环节：设计视频描述、生成视频片段、生成时长、结果选择、全屏预览、重新生成、编辑描述、替换、清空和高清入口。
  • 全局动作：定义批量生成分镜、批量生成视频片段、预览、进入剪辑的可交互条件与二次确认逻辑。

技术与执行

模块关系

梳理分镜、VO、视频描述之间的绑定关系，明确移动、复制、删除时是否连带旁白或视频结果。

数据保存

将对话上下文、图片、视频保存至项目；将剪辑结果、字幕、配音和导出结果沉淀到资产。

剪辑 iframe

梳理剪辑合成、字幕生成、旁白配音、素材管理、自动保存、导出下载等编辑器能力。

异常处理

为生成中状态、失败重跑、刷新后任务更新、清空与重新生成等场景预留交互和状态口径。

项目管理与优化

  • 将复杂视频生成拆成脚本、分镜、视频片段、剪辑导出四类核心对象，降低用户理解和研发实现成本。
  • 把 Agent 自动生成与人工修订放在同一条链路中，避免生成结果不可编辑、不可复用。
  • 在 PRD 中细化移动、复制、删除、合并、拆分等高频编辑动作，减少研发实现中的状态歧义。
