Business Case

Brand+ 电商图片处理平台

面向电商业务的内部图片处理平台，由 AI 设计中台提供底层 AI 能力支持。我负责把白底图、图片翻译、详情页多语种和卖点图等场景拆成可调用、可测评、可复用的产品能力。

电商图片白底图 / 图翻 / 详情页测评闭环

项目描述

Brand+ 的关键问题不是“有没有模型”，而是底层图像能力如何稳定进入真实电商生产链路。项目需要同时处理业务输入、工具唤起、生成结果、二次编辑、质量评估、历史任务和结果回传，最终支撑白底主图、多语言图片翻译、详情页处理、主图套图和卖点图等核心场景。

需求与设计

  • 白底图：设计图片上传、参数预填、白底图检测、通用抠图、背景色/尺寸设置、结果展示和继续编辑链路。业务唤起 iframe 时，可携带图片 URL、抠图参数和自动执行状态。
  • 图片翻译：拆解图片内容检测、语种判断、OCR / mask、图片消除、多语言翻译、内容诊断、AI 排版和协议转换，支持原语言与目标语言不同的翻译链路。
  • 详情页 / 卖点图：设计商品主图、已有详情图、PC / Mobile 详情图、卖点选择、参数补充、多语言生成和结果回传方式，把生成结果以画板形式进入编辑器。
  • 二次编辑：在超编底层能力上叠加业务定制功能，支持手动抠图、拼图、预设样式、配件图上传、图片优化和保存回传。

技术与执行

能力封装

将白底图检测、抠图、图翻、详情页生成等能力拆成清晰输入输出，便于业务平台通过 iframe / API 接入。

测评体系

围绕白底图、图翻、详情页等场景建立批量测试口径，覆盖生成质量、翻译质量、处理耗时和 bad-case。

协议链路

梳理图翻 JSON 协议、HTML 协议、超编协议之间的转换关系，保证生成结果可继续编辑。

Skill 化复用

把套图、详情页、PRD、会议纪要等能力沉淀为可复用 Skill，方便后续跨场景调用。

项目管理与优化

  • 对接业务、设计、算法和研发，拆解 P0 / P1 能力范围，明确白底图、图翻、详情页、卖点图等能力的开发优先级。
  • 结合简历中的项目口径，累计支撑 1000W+ 图片处理量；抠图准确率达 97%，多语言翻译优质率达 96%。
  • 将一次性模型调试经验沉淀为评测规则、Prompt、bad-case 和能力文档，降低后续能力接入成本。
