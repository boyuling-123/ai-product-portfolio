Skill Case

钉钉 MCP 描述生成器

把原始 MCP 能力和字段转成清晰的工具描述、标签和使用边界。

工作流逻辑

  • 输入：原始 MCP 名称、能力说明、参数字段、调用约束和失败案例。
  • 处理：拆解触发场景、边界、参数解释、反例和模型容易误判的地方。
  • 输出：更清晰的工具描述、标签、适用/不适用场景和可验证调用标准。
  • 目标：提高工具被模型正确调用的概率，减少调用错和调用不到。

展示方式

这个 Skill 不适合只放一张截图，最好展示“原始描述 - 优化后描述 - 为什么更容易被调用”的三段逻辑。

素材缺口

原始描述 / 优化后描述 / 标签 / 通过率变化
