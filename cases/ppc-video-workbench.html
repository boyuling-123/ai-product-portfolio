Business Case

PPC 视频生成工作台

面向商品短视频生产，把商品信息、脚本、分镜、分镜图、视频片段、剪辑合成、保存导出串成一个 Agent 可生成、用户可干预、结果可沉淀的工作台。

AI 视频脚本 / 分镜 / 视频片段可编辑工作台

主链路

商品输入商品图、链接、语言、类型、渠道尺寸
脚本生成视频概览、卖点、分镜描述、VO 旁白
分镜画布分镜参考图、生成模式、结果选择
视频片段画面描述、时长、生成结果、高清处理
剪辑导出合成、字幕、配音、素材库、保存下载

项目描述

PPC 视频生成工作台解决的是“AI 能生成视频，但业务生产过程不可控”的问题。方案把一次性对话生成拆为多个可编辑模块，让用户能在脚本、分镜、视频片段和剪辑环节介入，并让过程数据进入项目和资产体系。

需求与设计

  • 入口与入参：定义商品图片、商品链接、目标语言、视频类型、投放渠道及尺寸等字段，支持进入 Agent 后自动唤起固定任务消息并返回前端组件。
  • 脚本环节：设计视频概览、分镜描述、VO 旁白等模块，并支持编辑、重新生成、双语辅助和适配视频时长翻译。
  • 分镜环节：设计未生成 / 已生成分镜状态，支持分镜参考图、首帧参考、首尾帧参考、多图参考、重新生成、局部修改、替换、复制和删除。
  • 视频环节：设计视频描述、生成视频片段、生成时长、结果选择、全屏预览、重新生成、编辑描述、替换、清空和高清入口。
  • 全局动作：定义批量生成分镜、批量生成视频片段、预览、进入剪辑的可交互条件与二次确认逻辑。

技术与执行

模块关系

梳理分镜、VO、视频描述之间的绑定关系，明确移动、复制、删除时是否连带旁白或视频结果。

数据保存

将对话上下文、图片、视频保存至项目；将剪辑结果、字幕、配音和导出结果沉淀到资产。

剪辑 iframe

梳理剪辑合成、字幕生成、旁白配音、素材管理、自动保存、导出下载等编辑器能力。

异常处理

为生成中状态、失败重跑、刷新后任务更新、清空与重新生成等场景预留交互和状态口径。

项目管理与优化

  • 将复杂视频生成拆成脚本、分镜、视频片段、剪辑导出四类核心对象，降低用户理解和研发实现成本。
  • 把 Agent 自动生成与人工修订放在同一条链路中，避免生成结果不可编辑、不可复用。
  • 在 PRD 中细化移动、复制、删除、合并、拆分等高频编辑动作，减少研发实现中的状态歧义。

PPC 视频设计工作台
商品输入脚本生成分镜画布视频片段剪辑导出

主功能链路

  • 用户输入商品图、商品链接、目标语言、视频类型、投放渠道和尺寸后，系统生成视频脚本、分镜、分镜图和视频片段。
  • 工作台把自动生成拆成可编辑对象：脚本、VO 旁白、分镜描述、分镜图、视频描述、视频结果和剪辑项目。
  • 每一步都允许用户查看、编辑、重新生成、替换或清空，避免一次性生成黑盒。

新增设计与版本变更

  • 新增项目能力：项目列表、新建项目、编辑已有项目、保存项目、从项目继续编辑。
  • 新增从工作台进入视频剪辑 iframe 的链路，并将剪辑结果保存为项目资产。
  • 4.7 版本新增分镜画布与视频片段画布的完整编辑能力。

输入入参

  • 商品图片、商品链接、目标语言。
  • 视频类型、投放渠道、画面比例、时长。
  • 可选参考图、品牌元素、场景要求和补充提示词。

生成过程

  • 生成视频概览与核心卖点。
  • 生成分镜描述、分镜参考图和 VO 旁白。
  • 基于分镜图和视频描述生成视频片段。

模块规则

脚本与分镜

  • 脚本模块包含视频概览、卖点、分镜描述和 VO 旁白；支持编辑、重新生成、双语辅助和按时长适配。
  • 分镜模块区分未生成与已生成状态；支持参考图、首帧参考、首尾帧参考、多图参考、局部修改、替换、复制、移动和删除。
  • 分镜与 VO 存在绑定关系，移动或复制时需要同步更新绑定数量与顺序。

视频片段

  • 视频描述模块从分镜描述继承画面信息，支持编辑描述、生成片段、重新生成、替换、清空和高清处理。
  • 已生成视频片段支持全屏预览、选择结果、替换资源和进入剪辑。
  • 删除语义调整为清空，避免误删对象结构和关联关系。

全局操作

  • 批量生成分镜：当存在分镜/场景描述时可用；生成中对象跳过；支持选择仅生成未生成或全部重新生成。
  • 批量生成视频片段：当存在视频描述与分镜图时可用；支持二次确认和失败重跑。
  • 预览：至少有一个选中的视频结果时可用，按分镜顺序全屏预览。
  • 进入剪辑：至少有一个选中的视频结果时可进入视频剪辑 iframe。

剪辑、保存与资产

  • 进入剪辑前预处理：视频剪辑合成、字幕生成、旁白配音生成。
  • 编辑器能力：基础视频编辑、贴纸、文本、转场、字幕样式和素材管理。
  • 保存规则：对话数据、生成过程数据、图片和视频保存至项目；最终剪辑视频保存至资产。

交互组件与状态验收

  • 新增、复制、移动、删除需要区分分镜、VO、视频描述、已生成结果等不同对象。
  • 分镜移动到 VO 内外时，需要维护分镜与 VO 的绑定关系；多选移动/删除时按选中数量判断是否保留原 VO。
  • 生成中、生成失败、刷新返回、切换工具返回等状态，需要能重新拉取最新任务结果。
  • 多选和单选样式需要统一，避免用户误判操作对象。
